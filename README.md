# pAIr

Landing page for pAIr — an AI safety talent scoring and matching platform.

**Live:** [getpair.vercel.app](https://getpair.vercel.app)

## What is pAIr

pAIr conducts structured AI agent interviews with AI safety candidates, scores them against a field-built rubric, and generates a portable credential. Candidates take one interview and carry the score everywhere. Orgs skip redundant first-round screening and get a trusted signal they can act on.

## Stack

Single-file static HTML — Tailwind CDN, Google Fonts, vanilla JS.
